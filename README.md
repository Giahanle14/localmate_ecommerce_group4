# localmate_ecommerce_group4
LocalMate là website thương mại điện tử kết nối du khách với người bản địa cung cấp trải nghiệm du lịch. Hệ thống cho phép tìm kiếm, đặt tour và quản lý nội dung. Backend sử dụng PHP theo OOP kết hợp mô hình MVC, frontend dùng Bootstrap và tích hợp AJAX (JSON) để tối ưu trải nghiệm người dùng.
